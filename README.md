# Kotlin Language Study - CS420 Final Project

This repository contains the work for the CS420 Final Project, focusing on an in-depth study of the Kotlin programming language.

## Project Structure

- **`sample-programs/`** - Main sample program and feature extensions
- **`oop-programs/`** - Object-oriented test programs for inheritance and method dispatch
- **`report/`** - Written report documentation
- **`docs/`** - Additional documentation and research notes

## Quick Start

### Prerequisites
- Kotlin compiler (see report for setup instructions)
- Java Development Kit (JDK) 8 or higher

### Running the Sample Program
```bash
cd sample-programs
kotlinc -include-runtime HelloWorld.kt -d HelloWorld.jar
java -jar HelloWorld.jar
```
